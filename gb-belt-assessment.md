# Green Belt Assessment

## Green Belt Assessment @showdialog 
To show you are ready for Orange Belt, complete the Green Belt Mastery Mission to demonstrate your understanding of the concepts taught in Orange and Green Belts, including tilemaps, arrays, advanced conditionals, nested loops, functions, and project extensions like Tile Util, Scrolling Background, and Status Bar.

Create a project that fulfills all of the requirements on the left side. Click Next and alert a Code Sensei when you are ready to have your project checked.

Click **Ok** to get started!

## Requirements

A complete Belt Assessment must meet the following requirements: 

- :paint brush: Create at least 2 **tilemaps**. Sprites should be placed on each tilemap using specific coordinates, random tiles, and/or placeholder tiles.  
- :paper plane: In a 2D platformer tilemap, use **velocity** to allow the Player sprite to jump when the A button is pressed, and **acceleration** to create gravity that pulls a Player sprite down. Only allow the Player sprite to jump if the bottom of the sprite is touching a tilemap wall.
- :paper plane: Use **overlap events** to make things happen when a sprite overlaps a tile and a wall on the tilemap. 
- :id card: Use **data** to keep track of information in a project. Use **variables** to change elements such as speed, position, and true/false conditions. Use **arrays** to create, modify, and use lists of sprites, sprite images, strings, or numbers. Store **user input** in **variables** and **arrays** to be used in the project.
- :shuffle: Use **logic** and **Booleans** with **comparisons**, **conditionals**, and **variables** to make complex things possible, such as requiring 2 things to be true to win a game.
- :repeat: Use **loops** and **nested loops** to repeat code in a project. Use loops to create and/or modify multiple sprites, make something repeat while a certain condition is true, or to make something happen “forever” in a project.
- :function: Use **functions** to create sprites in a project. Use **functions** with **parameters** to customize it with number and Boolean values that are used to set the sprite’s properties.
- :plus: Use code from the **Tile Util**, **Scrolling Background**, and **Status Bar** extensions to add exciting elements to a project.
