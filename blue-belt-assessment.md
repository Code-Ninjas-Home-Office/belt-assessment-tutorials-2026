# Blue Belt Assessment

## Blue Belt Assessment @showdialog 
To show you are ready for Purple Belt, complete the final Prove Yourself of Blue Belt to demonstrate your understanding of the concepts taught in Blue Belt, including image and movement animations for sprites, looping melodies, and using the Scroller and Story extensions.

Create a project that fulfills all of the requirements on the left side. Click Next and alert a Code Sensei when you are ready to have your project checked.

Click **Ok** to get started! 

![Logo](https://github.com/Code-Ninjas-Home-Office/game-building-session-tutorials-2024/blob/master/images/CN-Logo.png?raw=true "CN Logo") 

## Requirements

A complete Belt Assessment must meet the following requirements: 

- :plus: Use the **Scroller** extension to create a scrolling background with at least 3 layers. Use a variable to set up each background image layer. Then, scroll each layer, except the bottom layer, at a different speed.
- :plus: Use the **Story** extension to provide the user with at least 1 choice of at least 3 different options that will change something in the project.
- :paint brush: Use **image** and **movement animation** to change the appearance of a sprite.
- :music note: Use a string to create a **looping melody** that plays in the background of the project.
